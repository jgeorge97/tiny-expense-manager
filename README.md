# Tiny Expense Manager

Tiny Expense Manager is a simple expense manager to keep track of earning's and expenses. All the data is stored in LocalStorage.

[Live Preview](https://tiny-expense-manager.netlify.app/)

## Technologies Used
- Vue 2
- Buefy
